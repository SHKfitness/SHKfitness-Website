# SHK Fitness Website

A responsive fitness website built with HTML, CSS, and JavaScript that can be deployed on GitHub Pages.

## Features

- Fully responsive design (mobile, tablet, desktop)
- Modern fitness-focused UI/UX
- Interactive elements (contact form, workout buttons)
- Smooth scrolling navigation
- Animated elements on scroll
- Contact information with SHKfitness@gmail.com

## Sections

1. **Navigation** - Fixed header with smooth scrolling links
2. **Hero** - Eye-catching introduction with call-to-action buttons
3. **Programs** - Four fitness program cards
4. **Workouts** - Interactive workout list with details
5. **Nutrition** - Fitness nutrition tips with icons
6. **About** - Information about SHK Fitness with statistics
7. **Contact** - Contact form and information
8. **Footer** - Additional links and newsletter signup

## How to Deploy on GitHub Pages

1. Create a new repository on GitHub named `SHKfitness-website`
2. Clone the repository to your local machine
3. Copy all files (index.html, style.css, script.js, README.md) to the repository folder
4. Create an `assets` folder for images (optional)
5. Commit and push your changes
6. Go to repository Settings > Pages
7. Under "Source", select "Deploy from a branch"
8. Select the main branch and root folder
9. Click Save
10. Your site will be published at `https://[your-username].github.io/SHKfitness-website/`

## Customization

- Replace placeholder icons with actual images in the `assets` folder
- Update color scheme in CSS `:root` variables
- Modify content in HTML files
- Update contact email to SHKfitness@gmail.com
